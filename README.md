# 🎨 Color Square Loader

A simple and modern **5-square colorful loading animation** built using **HTML & CSS**.
Perfect for adding a stylish loading effect to your website or project.

---

## 🚀 Features

* 🌈 Multi-color (RGB-style) animation
* 🟦 5 square blocks in a row
* 🔄 Smooth wave loading effect
* ⚡ Lightweight (no JavaScript needed)
* 🎯 Centered layout with clean design

---

## 📸 Preview

A row of 5 squares that animate in a wave pattern with changing colors on a white background.

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations & Keyframes)

---

## 📂 How to Use

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/color-square-loader.git
   ```

2. Open the project folder.

3. Run the `index.html` file in your browser.
